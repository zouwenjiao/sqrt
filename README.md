# sqrt

A GitHub repository for us to use to practice branching and merging. Developed for use in the UBC DSCI 524 - Collaborative Software Development course.

## Usage:

For example, to calculate the square root of 25:
```
python sqrt.py --n=25
```

```
5.0
```

## Dependencies:
- Python 3 or greater
- Python pacakges:
  - `docopt`
  - `math`
